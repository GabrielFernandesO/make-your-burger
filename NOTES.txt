PROJETO COM UM BACK-END FAKE

SEGUE A INSTALÇÃO:

npm install json-server (vai funcionar como um api) e deverá ser inicializado

ENTÃO BASICAMENTE O VUE SERÁ O FRONT
E ESSE JSON SERVER O NOSSO BACK END FAKE. QUE VAI SIMULAR NOSSO BACK END.

OBS: TEM QUE EXECUTAR O SCRIPT QUE CRIAMOS PARA O JSON SERVER npm run backend
EM PARALELO COM O VUE npm run dev. ISSO TUDO ESTÁ NAS DEPENDENCIAS DO PROJETO.

O BACK FUNCIONA EM PARALELO DO FRONT, POR ISSO A IDEIA SERÁ LINKAR O LOCALHOST DO BACK COM O FRONT.
